# Changelog

All notable changes to the Game Launcher Cloud - Manager for Godot will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-01-08

### Added
- Initial release of Game Launcher Cloud Manager for Godot
- API Key authentication
- App selection and management
- Build & upload functionality for Windows, Linux, and macOS
- Automatic build compression (ZIP)
- Real-time upload progress tracking
- Multipart upload support for large files (>500MB)
- Build status monitoring
- Configuration persistence
- Developer mode for testing with different environments
- Support for Godot 4.x
- Comprehensive documentation and quick start guide
